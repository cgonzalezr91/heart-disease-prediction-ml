# Predicción de Enfermedad Cardíaca ❤️

Proyecto de Machine Learning basado en el dataset Heart Disease UCI, enfocado en predecir la presencia de enfermedad cardíaca en pacientes.

## Objetivo
Aplicar la metodología CRISP-DM para desarrollar un modelo de clasificación que permita identificar la probabilidad de enfermedad en base a variables clínicas.

## Dataset
Incluye variables como:
- Edad
- Sexo
- Presión arterial
- Colesterol
- Frecuencia cardíaca

## Tecnologías
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Proceso
- Comprensión del negocio y datos
- Limpieza y preparación de datos
- Análisis exploratorio (EDA)
- Entrenamiento de modelo
- Evaluación de métricas

## Resultados
Se desarrolló un modelo capaz de clasificar la presencia de enfermedad, evaluado mediante métricas como accuracy, precision y recall.

## Conclusión
El proyecto demuestra cómo la analítica avanzada permite generar valor a partir de datos, aplicable en contextos como salud, finanzas y gestión de riesgos.
